# Servix CMMS - Hospital Maintenance Management System

![Servix CMMS Dashboard](https://placehold.co/800x400/14b8a6/ffffff?text=Servix+CMMS+Dashboard)

A full-stack Computerized Maintenance Management System (CMMS) designed for hospitals to manage, oversee, and optimize equipment performance.

## Features

### 🏥 Hospital Equipment Management
- Complete inventory tracking with status monitoring
- Department-based categorization (ICU, Radiology, Laboratory, etc.)
- Critical equipment identification
- QR code generation and scanning

### 🔧 Maintenance Management
- Preventive maintenance scheduling (calendar-based, usage-based)
- Corrective maintenance requests
- Maintenance history tracking
- Document and image attachments

### 📋 Task Management
- Engineer-to-technician task assignment
- Priority levels and deadlines
- Real-time status updates
- Progress tracking

### 📊 Dashboard & Reporting
- Equipment status overview
- Maintenance completion statistics
- Departmental asset counts
- Upcoming maintenance alerts
- Exportable reports (PDF/Excel)

### 🔐 Secure Authentication
- Role-based access control (Engineer, Technician)
- Secure password hashing
- JWT authentication
- Session management

## Technology Stack

### Frontend
- React.js with React Router
- Tailwind CSS for styling
- Axios for API calls

### Backend
- Node.js with Express.js
- MongoDB with Mongoose ODM
- JSON Web Tokens for authentication
- RESTful API design
